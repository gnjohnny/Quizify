# 🧠 Dynamic Trivia Quiz App

A modern, fast, and fully responsive Quiz Application built using vanilla web technologies. The application dynamically connects to a public trivia API to deliver an engaging, customizable quiz experience across multiple categories and difficulty levels.

## ✨ Features

- **Customizable Setup:** Choose your preferred category, question count, and difficulty (Easy, Medium, Hard) before diving in.
- **Dynamic API Integration:** Fetches real-time data with robust error handling and smooth loading states.
- **Active Gameplay Mechanics:** Features a visible countdown timer per question, an active progress bar, and instant visual feedback (green/red) upon selection.
- **Auto-Advance:** Automatically locks inputs and reveals the correct answer if the timer runs out.
- **Detailed Analytics:** A final results screen featuring a score percentage, personalized feedback, and a complete question-by-question breakdown of your choices versus the correct answers.
- **Polished UI/UX:** Built with a clean, minimalist aesthetic featuring fluid transitions, hover effects, and full responsiveness across mobile, tablet, and desktop viewports.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Modern variables / Tailwind CSS layout)
- **Logic:** Vanilla JavaScript (ES6+, asynchronous fetch API, modular state management)
- **Data Source:** Open Trivia DB / Public Trivia API

## ⚙️ Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/gnjohnny/open-trivia.git
   ```
2. Open `index.html` directly in your browser, or spin up a local server using an extension like Live Server.
3. Select your settings and test your knowledge!
